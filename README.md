# Kali Rocker Cuántico 🤘💀🎶

> **¡Personalizá tu Kali Linux como un verdadero Hacker Cósmico Rockero!**

---

## ⚙️ ¿Qué hace este script?

Este script instala una estética única, poderosa y visualmente explosiva sobre Kali Linux:

- 🎩 Oh-My-Zsh + tema `powerlevel10k`  
- 🎸 Banner cuántico "TINO ROCKER CUÁNTICO" en terminal con `figlet` y `lolcat`  
- 🧠 Efectos visuales: `neofetch` y `cmatrix` para esa vibra hacker rockera  
- 🎶 Música rockera que se reproduce automáticamente al iniciar sesión con `mpg123`  
- 🎨 Preparado para incluir fondos animados (opcional)  
- 🔧 Crea automáticamente la estructura de carpetas `/extras` para los archivos necesarios
